## 📋 Loyiha haqida

**AirPods Pro Landing Page** — Apple AirPods Pro quloqchinlarini professional tarzda taqdim etuvchi zamonaviy mahsulot sahifasidir. Minimal va elegant dizayn orqali foydalanuvchiga qurilma xususiyatlari, narxlar va sotib olish imkoniyati taqdim etiladi.
